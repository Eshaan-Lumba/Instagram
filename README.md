# Instagram

This is an Instagram clone with a custom Parse backend that allows a user to post photos, view a global photos feed, and add comments. 

User can register, log in and log out, and stay logged in across restarts. They can take a photo, add a caption and post it to the server. They can see the last 20 posts, view the comments on it and add their own comment. 

Made using: Swift, Storyboard, Parse
